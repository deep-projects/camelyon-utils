# CAMELYON Utils

This is a set of tools to process slides of the [CAMELYON](https://camelyon16.grand-challenge.org/) image database.


## Usage

```bash
camelyon-utils --version
camelyon-utils --help
camelyon-utils slide-tiles --help
```
